# FUTURE_CS_02

## Phishing Email Detection & Awareness System

### Internship
Future Interns – Cyber Security Internship

### Objective
To analyze phishing and suspicious email samples, identify phishing indicators, classify email risks, and provide awareness guidelines for users.

### Tools Used
- Gmail
- Google Admin Toolbox Message Header Analyzer
- MXToolbox Email Header Analyzer
- Canva
- GitHub

### Email Samples Analyzed
1. ZeroToFounder Email
2. Stable Money FD Rates Email
3. April 2026 Winners Email

### Analysis Performed
- Header Analysis
- SPF Verification
- DKIM Verification
- DMARC Verification
- Risk Classification
- Awareness Assessment

### Risk Classification
All analyzed emails were classified as Suspicious Emails with Medium Risk.

### Deliverables
- Phishing Detection Report
- Email Evidence Screenshots
- Header Analysis Evidence
- Awareness Guidelines
